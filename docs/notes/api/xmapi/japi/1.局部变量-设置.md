# 局部变量 - 设置
```jass
function XMSetLocalVariables takes integer frame returns integer
```
# 描述
设置局部变量
# 注意
不能穿透计时器和触发


# 参数
类型|名字|说明
--|--|--
整数|frame| 控件地址


# 返回值
类型|说明
--|--
整数| 控件地址


# 例子

```jass

local integer parent = FrameGetParent(frame)

```

```lua
local japi = require 'jass.japi'

local parent = japi.FrameGetParent(frame)

```
